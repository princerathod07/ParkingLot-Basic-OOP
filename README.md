# Parking Lot Management System

A console-based Parking Lot Management System developed in C++ that helps manage vehicle parking efficiently. The system allows users to park vehicles, remove vehicles, check parking status, and prevent duplicate entries.

## Features

- Park a vehicle
- Remove a parked vehicle
- View parking lot status
- Prevent duplicate vehicle entries
- Store and load vehicle data using file handling

## Technologies Used

- C++
- Object-Oriented Programming (OOP)
- File Handling
- Console-based Interface

## OOP Concepts Implemented

- Encapsulation
- Inheritance
- Polymorphism

## How It Works

1. User runs the program.
2. A menu is displayed with parking options.
3. User can park or remove vehicles.
4. System checks for duplicate entries.
5. Vehicle data is saved to a file for future use.
